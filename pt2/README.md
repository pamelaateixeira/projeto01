# pt2

gerenciar tarefas:

'bun cli.ts lista.jsom add "muahahaha"' - adiciona nova tarefa
'bun cli.ts lista.json list' - lista tudo
'bun cli.ts lista.json toggle 0' - conclui oq estava pendente(false)
'bun cli.ts lista.json update 0 "muehehehe"' - troca o nome de alguma tarefa
'bun cli.ts lista.json remove 0' - remove uma tarefa

To install dependencies:

```bash
bun install
```

To run:

```bash
bun run 
```

This project was created using `bun init` in bun v1.3.11. [Bun](https://bun.com) is a fast all-in-one JavaScript runtime.
