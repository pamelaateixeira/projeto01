# parte1

gerenciar tarefas:

'bun cli.ts add "muahahaha"' - adiciona nova tarefa
'bun cli.ts list' - lista tudo
'bun cli.ts toggle 0' - conclui oq estava pendente(false)
'bun cli.ts update 0 "muehehehe"' - troca o nome de alguma tarefa
'bun cli.ts remove 0' - remove uma tarefa

To install dependencies:

```bash
bun install
```

To run:

```bash
bun run 
```

This project was created using `bun init` in bun v1.3.11. [Bun](https://bun.com) is a fast all-in-one JavaScript runtime.
